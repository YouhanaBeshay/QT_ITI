# Qt Weather App

- This is a simple QT weather app made using the `open-meteo forecast API`.
- currently its staticly displaying the weather for Cairo. 
  - <small> can be modified to display the weather for any city using the `open-meteo Geocoding API` but not needed for this simple demonstration.</small>

## Features:

- Displays the current weather for Cairo.
- Displays the forecast for this week (7-day forecast).
- Displays the hourly forecast for current day 24 hours.
